# Competitor research

## How to use this folder

- **[`_patterns_catalog.md`](_patterns_catalog.md)** — 189 named UI patterns (adopt/avoid). Start here for any design decision.
- **[`_crosscut_metric_dictionary.md`](_crosscut_metric_dictionary.md)** — 85 canonical metric names + formulas. Use these; don't invent new ones.
- **`_ui_research_*.md`** — focused UI/UX breakdowns per competitor group (April 2026).
- **`_crosscut_*.md`** — cross-competitor deep dives on specific surfaces (pricing, onboarding, mobile, export, copy, multi-store).
- **`_inspiration_*.md`** — non-ecommerce UX references (Linear, Stripe, Vercel, Plausible, GA4).
- Individual files (amplitude, fairing, klaviyo, shopify-native, varos) — positioning, pricing, specific patterns.
- `_screens/` — reserved for screenshot files.
